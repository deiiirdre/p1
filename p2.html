Definition

Internet Protocol address (IP address) refers to the logical address of the host and the network in the Internet. It is a numberical label, and can block physical address differences.

Format

An IPv4 address has a total of 32 bits,which limits the address space to 4294967296 (232) addresses.An IPv4 address is usually represented in dotted decimal notation and consists of four decimal numbers, each ranging from 0 to 255, separated by dots, each representing a set of 8-bit (octets) addresses.
